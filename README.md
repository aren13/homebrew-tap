# homebrew-tap

Personal Homebrew tap for [@aren13](https://github.com/aren13).

## Install

```sh
brew tap aren13/tap
brew install aren13/tap/accountpilot
```

Or in one line:

```sh
brew install aren13/tap/accountpilot
```

## Available formulae

| Formula | Description | Upstream |
|---|---|---|
| `accountpilot` | Unified per-machine account sync framework — email, iMessage, calendar, and more | [aren13/accountpilot](https://github.com/aren13/accountpilot) |
